\# Hack and Slash Prototype



This repository contains a hack and slash prototype developed with \*\*Unity 6000.3.16f1\*\*.



\## Getting Started



1\. Install \*\*Unity 6000.3.16f1\*\* using Unity Hub.

2\. Clone or download this repository.

3\. Open \*\*Unity Hub\*\*.

4\. Click \*\*Add\*\* and select the project folder.

5\. Open the project using \*\*Unity 6000.3.16f1\*\*.

6\. The prototype can be played with keyboard and mouse or gamepad.
